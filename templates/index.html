Calculator API

A simple and powerful REST API for basic mathematical operations

Base Endpoint

GET /{operation}/{number1}/{number2}
Addition

Adds two numbers

Try /add/5/3
Subtraction

Subtracts two numbers

Try /subtract/10/4
Multiplication

Multiplies two numbers

Try /multiply/6/7
Division

Divides two numbers

Try /divide/15/3

Response Format

{ "status": 200, "result": 8 }
Note: All operations return JSON responses. Error responses will include appropriate status codes and error messages.
